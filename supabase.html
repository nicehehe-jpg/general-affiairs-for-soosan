Supabase 연결 설정

데이터를 클라우드에 저장해 PC가 바뀌어도 그대로 사용할 수 있습니다.

🔑 프로젝트 연결 정보

👉 Supabase 대시보드 → 프로젝트 선택 → Settings → API 에서 확인할 수 있습니다.

🛠️ 테이블 생성 (최초 1회)

Supabase 대시보드 → SQL Editor → New Query 에 아래 SQL을 붙여넣고 실행하세요.

-- 1. 할 일(Todo) 테이블 create table if not exists todos ( id text primary key, title text, memo text, priority text default 'medium', due_date text, is_completed boolean default false, completed_at timestamptz, tags jsonb default '[]', sub_tasks jsonb default '[]', cat text, created_at timestamptz default now(), updated_at timestamptz default now() ); -- 2. 월간 업무 테이블 create table if not exists gongmu_tasks ( id text primary key, cat text, title text, priority text, status text default 'wait', due text, created_at timestamptz default now(), updated_at timestamptz default now() ); -- 3. 주소록 테이블 create table if not exists contacts ( id bigserial primary key, name text, email text, company text, department text, title text, phone text, mobile text, created_at timestamptz default now() ); -- RLS 비활성화 (내부 사용용) alter table todos disable row level security; alter table gongmu_tasks disable row level security; alter table contacts disable row level security;
